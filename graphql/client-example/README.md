# React + Apollo Client Example

## Setup (in your React project)

```bash
npm install @apollo/client graphql
```

## Files to add

- `apolloClient.js` – Apollo Client setup
- `App.jsx` – Example component that fetches users from your GraphQL API

## Run

1. Start your GraphQL backend: `cd .. && npm start` (runs on http://localhost:3000)
2. Start your React app
3. The example expects the GraphQL API at `http://localhost:3000`
